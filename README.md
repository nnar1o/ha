# Instrukcja obsługi repozytorium dodatków Home Assistant

Repozytorium to zawiera dodatki do Home Assistant. Jest gotowe do rozwoju i rozbudowy o kolejne dodatki.

## Dostępny dodatek

### sms-gateway
- **Funkcja:** Wysyłanie i odbieranie SMS przez modem Huawei
- **Obsługa:** MQTT

Zachęcamy do eksploracji i wkładu w rozwój repozytorium!
